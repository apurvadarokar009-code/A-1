Software Development Life Cycle

Stages:

Requirement analysis

Detailed analysis of all requirements is performed to understand and document what the software needs to do.

Planning

Involves defining project goals, scope, and resource requirements, and creating an initial project plan.

Design

The system architecture and high-level design are created, including defining hardware and software components and user interface design.

Development(coding)

The actual software code is written based on the design specifications.

Testing

The software is thoroughly tested to identify and fix bugs. This can include unit, integration, and user acceptance testing.

Deployment

The finished software is released and installed into the production environment, making it available to end-users.

Maintenance

Includes ongoing updates, bug fixes, and improvements to the software after it has been deployed.
